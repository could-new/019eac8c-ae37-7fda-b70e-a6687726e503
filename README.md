# احمد کی کہانی

یہ ایک سادہ اور خوبصورت فلٹر (Flutter) ایپ ہے جو ایک غریب مگر محنتی لڑکے، احمد کی کہانی سناتی ہے۔ اس ایپ میں اردو زبان کی سپورٹ (RTL - Right to Left) کو مدنظر رکھ کر ایک بہترین اور آرام دہ یوزر انٹرفیس (UI) بنایا گیا ہے۔

## خصوصیات
*   **اردو متن کی مکمل سپورٹ:** دائیں سے بائیں (RTL) زبان کے لیے درست الائنمنٹ۔
*   **خوبصورت انٹرفیس:** پڑھنے میں آسان اور دلکش ڈیزائن تاکہ کہانی پڑھتے وقت آنکھوں کو سکون ملے۔
*   **تمام ڈیوائسز کے لیے سازگار:** موبائل، ٹیبلیٹ، اور ویب پر بہترین ڈسپلے۔

## کیسے چلائیں (How to Run)
اس ایپ کو چلانے کے لیے آپ کے پاس Flutter کا سیٹ اپ ہونا ضروری ہے۔

1.  ریپوزٹری کلون کریں یا ڈاؤنلوڈ کریں۔
2.  ٹرمینل میں اس پروجیکٹ کے فولڈر میں جا کر درج ذیل کمانڈز چلائیں:
    ```bash
    flutter pub get
    flutter run
    ```
